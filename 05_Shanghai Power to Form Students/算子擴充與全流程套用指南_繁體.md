# 算子增加範例(繁體)

改 4 個檔,新算子就套用到全流程(05)。以下用 `twist`(權力漩渦)為例,照抄改名即可。

---

## 契約

```python
rec = {"geom": <shapely, 米 EPSG:32651>, "h": <float 米>,
       "sh": "state|developer|resident|unknown|informal", "frozen": <bool>}
```

- 純函數 `recs -> recs`;先複製 `out=[dict(r) for r in recs]`,別動原始。
- 只動 `r["sh"] in target and not r.get("frozen")`。
- GFA ∝ `geom.area × h`。守恆=縮面積補高度;加量=只抬高度。

---

## ① `engine/operators.py` — 貼函數 + `OPS` 加一行

```python
def twist(recs, target, deg_per_100m=12.0):
    """權力漩渦:目標樓 footprint 依到全域 GFA 重心的距離旋轉(越遠轉越多)。
    旋轉不改面積、高度不變 → 單棟 GFA 守恆。"""
    import math
    import shapely.affinity as aff
    out = [dict(r) for r in recs]
    A = sum(r["geom"].area * r["h"] for r in out) or 1.0
    Px = sum(r["geom"].centroid.x * r["geom"].area * r["h"] for r in out) / A
    Py = sum(r["geom"].centroid.y * r["geom"].area * r["h"] for r in out) / A
    for r in out:
        if r["sh"] in target and not r.get("frozen"):
            d = math.hypot(r["geom"].centroid.x - Px, r["geom"].centroid.y - Py)
            r["geom"] = aff.rotate(r["geom"], deg_per_100m * d / 100.0, origin="centroid")
    return out
```

檔尾:

```python
OPS = {..., "scale_graded": scale_graded, "twist": twist}   # ← 加 twist
```

> 只 `ops.register("twist", twist)` = runtime 登記,06 是獨立進程 → `KeyError`。必須永久寫進 `OPS`。
